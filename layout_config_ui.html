Step 1 of 7
Files & wages
Set the input Excel filename, output CSV filename, and the operator wage used to compute walking costs.
Input / output files
Exported from AutoCAD via the EXPORT_BLOCKS LISP script. Include the .xlsx extension.
Operator wage — used to compute weekly walking cost (MAD·cm / week)
Weekly wage = 17.92 × 7.67 × 5.75 = 790.34 MAD / week
Step 2 of 7
Zone coordinates
Copy Xmin / Xmax / Ymin / Ymax from AutoCAD (click zone rectangle → Properties panel). All values in centimetres.
Zone A — left red area (MECAL, HANKE)
Zone FREE — green middle area (high-interaction machines)
Zone B — right blue area (to evacuate)
Poutre (structural beam) — hard obstacle, 65 cm gap enforced from all machines

Production rates
Wire throughput (fils/hr) per machine. Used to weight which machine families should be physically close. Set 0 for inactive or offline machines.

Step 6 of 7
Relationship chart (REL)
How important is it for each pair of machine families to be physically close? 7 = must be adjacent 5–6 = very desirable 1–3 = nice to have 0 = neutral
